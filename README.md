# Maziyar Kolagar - Portfolio

A modern, bilingual (English/Persian) portfolio website showcasing my work as an AI Engineer and Web Developer.

**Live Site:** [https://thrymheim.github.io/Portfolio](https://thrymheim.github.io/Portfolio)

## Features

- **Bilingual Support** - Toggle between English and Persian (Farsi) with RTL layout
- **Responsive Design** - Fully responsive across all devices
- **Animated UI** - Particle effects, typing animation, scroll reveals, and counter animations
- **Contact Form** - Functional contact form powered by Formspree
- **Modern Dark Theme** - Clean, professional design with gradient accents

## Sections

- **Hero** - Introduction with animated typing effect
- **About** - Bio, profile image, and stats
- **Skills** - Web Development, AI/ML, and Tools
- **Projects** - Featured work with live demos and GitHub links
- **Contact** - Contact form and social links

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter, Fira Code, Vazirmatn)

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Thrymheim/Portfolio.git
   ```
2. Open `index.html` in your browser
